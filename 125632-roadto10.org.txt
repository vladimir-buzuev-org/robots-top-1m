	<span id="sectionPageBody">
<p><img alt="" src="/Medicare/Coding/ICD10/Other-Content-Types/ICD-10-Logo.jpg" /></p>
<p><b>CMS Releases ICD-10 Assessment and Maintenance Toolkit</b></p>
<p>The <a href="/Medicare/Coding/ICD10/Downloads/ICD-10NextStepsToolkit20170324.pdf">ICD-10 Next Steps Toolkit</a> gives providers the tools and information to assess ICD-10 progress using key performance indicators. With the Toolkit, providers can identify and address opportunities for improvement and maintain progress on ICD-10. The Toolkit is also available as an <a href="/Medicare/Coding/ICD10/Downloads/ICD10NextStepsInfographic20160303.pdf">infographic</a> with an accompanying <a href="/Medicare/Coding/ICD10/Downloads/ICD10KPIs20160309.pdf">Key Performance Indicators fact sheet</a>.</p>
<p><b>CMS Issues an ICD-10 Resource to Assist Providers in Finding Answers to their ICD-10 Questions</b></p>
<p>Recognizing that health care providers may need help with the ICD-10 transition, CMS has issued a <a href="/Medicare/Coding/ICD10/ICD-10-Provider-Contact-Table.pdf">resource guide and contact list</a>.</p>
<p><b>Medicare Fee-for-Service (FFS) Frequently Asked Questions</b></p>
<p>If you have Medicare FFS ICD-10 questions, please visit <a href="https://www.cms.gov/Medicare/Coding/ICD10/Frequently-Asked-Questions.html">Medicare FFS FAQs</a> for additional guidance.</p>
<p><b>Use ICD-10 Now!</b></p>
<p><b>ICD-10 Implementation Date: October 1, 2015</b></p>
<ul>
<li>Code services provided <span style="text-decoration: underline;">on or after</span> Oct 1, 2015&nbsp;with ICD-10</li>
<li>Code services provided <span style="text-decoration: underline;">before</span> Oct 1, 2015&nbsp;with ICD-9, even if you submit the claim after Oct 1, 2015</li>
<li>Medicare FFS <a href="https://questions.cms.gov/faq.php?id=5005&amp;faqId=12609">Guidance</a> for claims that span the compliance date and <a href="https://www.cms.gov/Medicare/Coding/ICD10/Frequently-Asked-Questions.html">FAQs</a> on claims processing and billing are available</li>
</ul>
<p>The ICD-10 transition is a mandate that applies to all parties covered by HIPAA, not just providers who bill Medicare or Medicaid.</p>
<p>2018 <a href="https://www.cms.gov/Medicare/Coding/ICD10/2018-ICD-10-CM-and-GEMs.html">ICD-10-CM</a> and <a href="https://www.cms.gov/Medicare/Coding/ICD10/2018-ICD-10-PCS-and-GEMs.html">ICD-10-PCS</a> files including General Equivalence Mappings are available.</p>
<p><b>Need Some Tips?</b></p>
<ul>
<li>Check out the&nbsp;<a href="/Medicare/Coding/ICD10/Downloads/UseICD-10NowHeresHow20170324.pdf">Use ICD-10 Now </a>and <a href="/Medicare/Coding/ICD10/Downloads/CMSICD10BillingSuccessInfographic.pdf">Billing Success</a> infographics</li>
<li>See listings of <a href="/Medicare/Coding/ICD10/Downloads/ICD10ResourcesFlyer20151007.pdf">CMS ICD-10 Resources</a> and <a href="/Medicare/Coding/ICD10/ICD-10-Coding-Resources.pdf">Coding and Clinical Documentation Resources</a></li>
<li>Get going with our <a href="/Medicare/Coding/ICD10/Downloads/ICD-10QuickStartGuide20170324.pdf">ICD-10 Quick Start Guide</a></li>
</ul>
<p><b>CMS-AMA Flexibility Guidance</b></p>

function XHConn()
{
  var xmlhttp, bComplete = false;
  try { xmlhttp = new ActiveXObject("Msxml2.XMLHTTP"); }
  catch (e) { try { xmlhttp = new ActiveXObject("Microsoft.XMLHTTP"); }
  catch (e) { try { xmlhttp = new XMLHttpRequest(); }
  catch (e) { xmlhttp = false; }}}
  if (!xmlhttp) return null;
  this.connect = function(sURL, sMethod, sVars, fnDone)
  {
    if (!xmlhttp) return false;
    bComplete = false;
    sMethod = sMethod.toUpperCase();
    try {
      if (sMethod == "GET")
      {
        xmlhttp.open(sMethod, sURL+"?"+sVars, true);
        sVars = "";
      }
      else
      {
        xmlhttp.open(sMethod, sURL, true);
        xmlhttp.setRequestHeader("Method", "POST "+sURL+" HTTP/1.1");
        xmlhttp.setRequestHeader("Content-Type",
          "application/x-www-form-urlencoded");
      }
      xmlhttp.onreadystatechange = function(){
        if (xmlhttp.readyState == 4 && !bComplete)
        {
          bComplete = true;
          fnDone(xmlhttp);
        }};
      xmlhttp.send(sVars);
    }
    catch(z) { return false; }
    return true;
  };
  return this;
}
