<p><b>Cloud VPS</b> (Virtual Private Server) provides a cloud computing infrastructure powered by <a href="https://en.wikipedia.org/wiki/OpenStack" class="extiw" title="en:OpenStack">OpenStack</a> for projects related to the <a href="https://meta.wikimedia.org/wiki/Wikimedia_movement" class="extiw" title="meta:Wikimedia movement">Wikimedia movement</a>. The environment includes access to a variety of <a href="/wiki/Portal:Data_Services" title="Portal:Data Services">data services</a>. Cloud VPS  is meant to make it easier for developers and system administrators to try out improvements to Wikimedia infrastructure (including MediaWiki), power research and analytics, and host projects that are not viable in the <a href="/wiki/Portal:Toolforge" title="Portal:Toolforge">Toolforge</a> environment.  Virtual machines running on Cloud VPS are subject to the <a href="/wiki/Portal:Wikimedia_VPS/instance_lifecycle" class="mw-redirect" title="Portal:Wikimedia VPS/instance lifecycle">instance lifecycle</a>.
</p>

</p>
