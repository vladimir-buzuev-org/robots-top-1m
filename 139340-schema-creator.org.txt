    <h1>Schema Structured Data</h1>
<hr>    
    <div class="addthis_jumbo_share"></div>
<hr>

<h2>What is Schema.org structured data?</h2>
<p><a href="/marketing-glossary/structured-data/">Structured data</a> is a way for search engine machines to make sense of content in your <abbr title="Hypertext Markup Language">HTML</abbr>. Google and other search engines created a structured data standard called <a href="http://schema.org/">Schema.org</a>.</p>
<h3>Schema creator tools</h3>
<p>When Schema.org was originally released, the core way of including it in your pages was to use microdata inside your <abbr title="Hypertext Marketup Language">HTML</abbr> elements. The Schema.org vocabulary and method of using microdata attributes was foreign to most web designers. That&#8217;s why we created the Schema Creator tool, to help web designers better understand and utilize Schema.org.</p>
<p>Using microdata attributes seemed elegant at the time, because it meant that you could markup your existing HTML without changing the content or appearance of the pages. It still wasn&#8217;t perfect though, and it could become cumbersome to integrate depending on how your pages were originally coded. Fortunately, there is now a simpler and better solution for integrating Schema.org structured data. That solution is JavaScript Object Notation for Linked Data (<abbr title="JavaScript Object Notation for Linked Data">JSON-LD</abbr>), which is basically JavaScript that you can place <em>anywhere</em> on your page.</p>
<p>Here&#8217;s a code example of Schema <abbr title="JavaScript Object Notation for Linked Data">JSON-LD</abbr>:</p>
<blockquote>
<p>&lt;script type=&#8217;application/ld+json&#8217;&gt;<br />
                    &nbsp;&nbsp;&nbsp;&nbsp;{<br />
                    &nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&#8220;@context&#8221;: &#8220;http://www.schema.org&#8221;,<br />
                    &nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&#8220;@type&#8221;: &#8220;product&#8221;,<br />
                    &nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&#8220;brand&#8221;: &#8220;Raven Tools&#8221;,<br />
                    &nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&#8220;name&#8221;: &#8220;Site Auditor&#8221;,<br />
                    &nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&#8220;image&#8221;: &#8220;https://raventools.com/img/og-raven.png&#8221;,<br />
                    &nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&#8220;description&#8221;: &#8220;Site Auditor analyzes your site to find all of the SEO issues that may be keeping your site from ranking on search engines.&#8221;<br />
                    &nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;}<br />
                    &lt;/script&gt;</p>
</blockquote>
<p>For the uninitiated, writing Schema structured data in <abbr title="JavaScript Object Notation for Linked Data">JSON-LD</abbr> can be intimidating. Fortunately, there are a couple of great tools that can make it significantly easier for you to implement <abbr title="JavaScript Object Notation for Linked Data">JSON-LD</abbr>.</p>
<h4><abbr title="JavaScript Object Notation for Linked Data">JSON-LD</abbr> Schema Generator</h4>
<p>Following in the footsteps of the original Schema Creator tool – the free tool Raven made that generated microdata for a select group of commonly used <em>types</em> – Hall Analysis has created the <a href="https://hallanalysis.com/json-ld-generator/"><abbr title="JavaScript Object Notation for Linked Data">JSON-LD</abbr> Schema Generator</a>. It too allows you to pick from a group of popular <em>types</em>, but instead of outputting microdata, it generates <abbr title="JavaScript Object Notation for Linked Data">JSON-LD</abbr>.</p>
<p><a href="https://hallanalysis.com/json-ld-generator/"><img src="data:image/gif;base64,R0lGODdhAQABAPAAAP///wAAACwAAAAAAQABAEACAkQBADs=" data-lazy-src="/img/seo-guide/json-ld-schema-generator.png" alt="JSON-LD Schema Generator" title="Schema generator for a Product type"><noscript><img src="/img/seo-guide/json-ld-schema-generator.png" alt="JSON-LD Schema Generator" title="Schema generator for a Product type"></noscript></a></p>
<h4>Schema App</h4>
<p><a href="https://www.schemaapp.com">Schema App</a> is a more full-featured paid app that helps you create and manage all of your structured data in one place. The tool allows you to pick from any <em>itemtype</em> and enter the data for it. You can then export it to multiple formats, including <abbr title="JavaScript Object Notation for Linked Data">JSON-LD</abbr>. They also have a <a href="https://wordpress.org/plugins/schema-app-structured-data-for-schemaorg/">WordPress plugin</a>.</p>
<p><a href="https://www.schemaapp.com"><img src="data:image/gif;base64,R0lGODdhAQABAPAAAP///wAAACwAAAAAAQABAEACAkQBADs=" data-lazy-src="/img/seo-guide/schema-app.png" alt="Schema App" title="Schema creator that manages and outputs structured data"><noscript><img src="/img/seo-guide/schema-app.png" alt="Schema App" title="Schema creator that manages and outputs structured data"></noscript></a></p>
