<p>In my spare time I have been working on my latest project <a href="http://1kgmore.org" target="_blank">1KG More</a> — a website built exclusively for nonprofit organizations that want to minimize operating cost. 1KG More allows nonprofit organizations to post project destinations, along with a list of items that could be used or are needed for the project.
