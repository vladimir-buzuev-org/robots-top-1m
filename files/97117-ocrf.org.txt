		<h2 class="block__title">Risk Factors</h2>
		<div class="block__content">

			<p>
				Many factors can increase or decrease a woman's risk of
				developing ovarian cancer.
			</p>

			<div class="factors factors--increase js-match-height">
				<h3 class="factors__title"><strong>Increases</strong> Risk</h3>
				<ul class="factors__list">
					<li class="factor factor--history">
						<span class="factor__icon"></span>
						<span class="factor__label">Family history of breast, ovarian or colon cancer</span>
					</li>
					<li class="factor factor--mutations">
						<span class="factor__icon"></span>
						<span class="factor__label">Genetic mutations like BRCA</span>
					</li>
					<li class="factor factor--post">
						<span class="factor__icon"></span>
						<span class="factor__label">Post-menopausal</span>
					</li>
					<li class="factor factor--age">
						<span class="factor__icon"></span>
						<span class="factor__label">Increased age</span>
					</li>
				</ul>
				<div class="factors__arrow"></div>
			</div>

			<div class="factors factors--decrease js-match-height">
				<h3 class="factors__title"><strong>Decreases</strong> Risk</h3>
				<ul class="factors__list">
					<li class="factor factor--pregnancy">
						<span class="factor__icon"></span>
						<span class="factor__label">Pregnancy</span>
					</li>
					<li class="factor factor--breastfeeding">
						<span class="factor__icon"></span>
						<span class="factor__label">Breastfeeding</span>
					</li>
					<li class="factor factor--contraceptives">
						<span class="factor__icon"></span>
						<span class="factor__label">Use of oral contraceptives</span>
					</li>
				</ul>
				<div class="factors__arrow"></div>

		<h2 class="block__title">Know the Symptoms</h2>
		<div class="block__content">

			<ul class="symptoms">
				<li class="symptom symptom--bloating">
					<span class="symptom__icon"></span>
					<span class="symptom__label">Bloating</span>
				</li>
				<li class="symptom symptom--eating">
					<span class="symptom__icon"></span>
					<span class="symptom__label">Difficulty Eating</span>
				</li>
				<li class="symptom symptom--pain">
					<span class="symptom__icon"></span>
					<span class="symptom__label">Pelvic / Abdominal Pain</span>
				</li>
				<li class="symptom symptom--urinary">
					<span class="symptom__icon"></span>
					<span class="symptom__label">Urinary Frequency</span>
				</li>
			</ul>
			<p>
				If these symptoms occur for <strong>more than 2 weeks</strong>
				and these symptoms are new or unusual for you, see a
				gynecologist and ask about ovarian cancer. Research shows that
				seeing a gynecologic oncologist for surgery and treatment
				significantly improves outcomes.
			</p>
