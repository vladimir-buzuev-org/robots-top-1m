<h1>Willkommen auf Skoom.de</h1>
<p>
	
	Skoom.de existiert bereits seit dem Jahr 2000. Mittlerweile kann man hier 
	eine <a href="./toplist-script/">PHP Topliste</a> downloaden und findet ausserdem <a href="./tutorials/">Tutorials</a> zu diversen Themen des 
	Webmasterlebens. Die Tutorials umfassen Themen wie <a href="./tutorials/suchmaschinenoptimierung/">Suchmaschinenoptimierung</a>, <a href="./tutorials/regexp/regulaere-ausdruecke.xtc">Reguläre Ausdrücke</a>,  <a href="./tutorials/php/">PHP</a> und  <a href="./tutorials/javascript/">JavaScript</a>. 
	Wenn Sie sich mit anderen Webmastern betreffend der Tutorials oder der Topliste austauschen möchten können Sie dies in dem <a rel="nofollow" href="/board/">Forum</a>von skoom.de tun.
	</p>
<p>
	Neben den Tutorials und der Topliste bieten wir seit 17.6.2006 einen völlig <a href="./gratis-counter/">kostenlosen Counter</a> an. Dieser funktioniert ohne Anmeldung sodass er nur kopiert und eingefügt werden muss.
	Danach ist er sofort einsatzbereit und beginnt zu zählen. Die IP-Sperre beträgt 60 Minuten und alle Grafikcounter aktualiseren sich im 20-Minuten-Takt. Wenn Sie einen <a href="http://www.counter.de/" target="_blank">Counter</a> mit Statistiken suchen, werden Sie sicher bei Counter.de fündig werden.</p>

<h2>SEO Tools</h2>
<p>SEO bedeutet Suchmaschinenoptimierung (engl. = <span style="border-bottom: Solid 1px #00AA00">S</span>earch <span style="border-bottom: Solid 1px #00AA00">E</span>ngine <span style="border-bottom: Solid 1px #00AA00">O</span>ptimization). Optimierung ist sehr wichtig für jeden erfolgreichen Webauftritt. Eine neue Webseite hat einen <a href="./service/keyworddichte-pagerank-ermitteln.xtc">Pagerank&#8482;</a> von 0 wodurch Sie von Suchmaschinen so schnell wohl nicht in den Index aufgenommen wird. Wir erklären einerseits wie man sich einen akzeptablen Pagerank aufbauen kann, sodass man von Suchmaschinen indiziert wird andererseits wie man seine Webseite so optimiert das man bei Suchmaschinen besser gefunden wird.<br/>Des Weiteren bieten wir <a href="./service/keyworddichte-pagerank-ermitteln.xtc">Tools</a> an mit denen Sie Ihren Pagerank&#8482; überprüfen können oder Ihre Keyworddichte analysieren können.</p>
