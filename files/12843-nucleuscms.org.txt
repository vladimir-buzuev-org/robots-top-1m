<h3><a href="http://www.nucleuscms.org/item/3063" title="Nucleus CMS is Back (Sort of)">Nucleus CMS is Back (Sort of)</a></h3>
Despite project close announcement one and a half years ago, some enthusiastic Japanese developers and users have been maintaining Nucleus CMS core.<br />
And today, we are pleased to announce Nucleus CMS 3.70. This is a direct descent of the original Nucleus CMS, supports the same multiple languages.<br />
Major fixes and improvements in this release are including (but not limited to) the following:<br />
<ul>
<li>Enforced password security using phpass library instead of md5 hash</li>
<li>Default character code set is now utf-8, not iso8859-1</li>
<li>Added new event PreLoadMainLibs</li>
</ul>
Upcoming version 3.71 is pre-released in Japanese as well. New features in 3.71 are:<br />
<ul>
<li>Fixed vulnerability described in <a href="http://cve.mitre.org/cgi-bin/cvename.cgi?name=CVE-2011-3760">VCE-2011-3760</a>.</li>
<li>Support for PHP7 (Fixes in installer, embedded wrapper for mysql functions, etc.)</li>
<li>[New] Duplicate Item button in the item list screen</li>
<li>[New] Now you can arrange category order</li>
<li>... and more!</li>
</ul>

Nucleus CMS 3.70 is available for download from <a href="http://www.nucleuscms.org/download.php">download page</a>.<br />
<br />
You are always welcome to post suggestions and questions for Nucleus CMS in the same old <a href="http://www.nucleuscms.org/forum/">English support forum</a>.<br />
Also, all the core and major plugins are now maintained on <a href="https://github.com/NucleusCMS">GitHub</a>. Please feel free to submit issues and of course pull requests.<br />
<br />
Beside this code base, LMNucleus a fork from Nucleus by Leo is also available at <a href="http://nucleus.slightlysome.net/">http://nucleus.slightlysome.net/</a> and <a href="https://github.com/slightlysome/lmnucleuscms">GitHub</a>.<br />
<br />
Cheers, and happy blogging!<br />
The Japanese Nucleus Team

January 27, 2016 - <a href="http://www.nucleuscms.org/item/3063">Permalink</a>

<h3><a href="http://www.nucleuscms.org/item/3061" title="Nucleus CMS 3.65 Released">Nucleus CMS 3.65 Released</a></h3>
<p>The Nucleus CMS core development team has released Nucleus CMS v3.65. This release includes a few minor bug fixes that are intended to make it compatible with PHP 5.4. <b>It is recommended that only users who are having issues with PHP5.4 support upgrade at this time. There are some bugs with character encoding in the Admin area.</b></p>
The change log can be found <a href="http://nucleuscms.org/documentation/history.html">here</a>.</p>
