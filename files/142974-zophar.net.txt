"vDosPlus (formerly vDos-lfn) is a general-purpose non-gaming DOS emulator running on Windows, based on vDos and DOSBox. vDosPlus is designed to run non-gaming DOS applications in all current Windows versions. However, most today's Windows is now 64-bit, and to run DOS programs you need a DOS emulator. vDosPlus is exactly for this purpose, and like original vDos it provides advanced features such as support for file-locking (needed by multi-user applications) and printing to Windows-only (GDI) printers, but it also has many unique features such as a tighter integration with Windows and enhanced keyboard, mouse and display support."</p>
