<h3>Kein Javascript aktiv</h3>
<div class="content">
<strong>Bitte aktivieren Sie Javascript in Ihrem Browser</strong>
<p>
Um alle Funktionalitäten auf JustMusic nutzen zu
können, benötigen Sie aktiviertes Javascript in Ihrem Browser.
</p>
<h4>Aktivieren Sie Javascript:</h4>
<h5>Internet Explorer:</h5>
<p>
Klicken Sie auf "Extras"-"Internetoptionen"-"Sicherheit"-"Stufe
anpassen" - "Scripting" und aktivieren Sie "Active Scripting".
</p>
<h5>Mozilla Firefox:</h5>
<p>
Klicken Sie auf "Extras"-"Einstellungen"-"Inhalt"-"Javascript
aktivieren". Wenn Sie Firefox 3.0 verwenden, klicken Sie bitte auf
"Bearbeiten"-"Einstellungen"-"Inhalt"-"Javascript aktivieren".
</p>
<h5>Google Chrome:</h5>
<p>
Klicken Sie auf das Symbol "Google Chrome anpassen" in der rechten
oberen Ecke Ihres Browsers und wählen Sie "Einstellungen". Klicken
Sie anschließend im unteren Bereich des Fensters auf "Erweiterte
Einstellungen anzeigen...". Öffnen Sie das Dialog-Fenster
"Inhaltseinstellungen" durch Klick au den gleichnamigen Button.
Wählen Sie unter der Überschrift "<strong>JavaScript</strong>"
die Option "Ausführung von JavaScript für keine Webite zulassen.".
</p>
<strong>Javascript aktiviert?</strong>
<p>
Nachdem Sie Javascript aktiviert haben, klicken Sie
<a href="/de-de">hier</a>!
</p>
